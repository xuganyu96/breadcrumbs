# Immutable lists, sets, and dictionaries

## Immutable list API
**`ImmutableList.append(obj)`**
* Return a copy of the list that contains everything of the original list as well as the new object at the end of the list

**`ImmutableList.insert(index, obj)`**
* Return a copy of the original list where the new obj is inserted at the specified index and all subsequent elements are moved to the right

**`ImmutableList.pop`**
* Return a tuple of two: `obj` and `l`. `obj` is the last element of the original list, and `l` is a copy of the original list but with the last element removed

**`ImmutableList.copy`**
* Return a copy of the list
